各ファイルの説明

・RTC
　宅配受付システムで利用するRTコンポーネント群

    ・Scale_EJ
 　  EJ4100 から質量を取得するコンポーネント

    ・Realsense
 　  Intel Realsense R200 からカラー画像・深度画像を取得するコンポーネント

    ・Dim_measure
　   入力された画像から荷物の寸法を計算するコンポーネント

    ・Package_consolidation
     荷物の情報を統合するコンポーネント

    ・DeliveryManager
     宅配受付システムの統合管理とユーザーからの情報入力を受け付けるコンポーネント

    ・DeliveryDatabaseRTC
     荷物の寸法,重さ,宅配方法,送り先住所,送り元住所から料金,お届け日数を計算する
コンポーネント

    ・PrintDeliveryRTC
     入力された情報を伝票に印刷するコンポーネント

・Document
　 各種マニュアルが入ったファイル
